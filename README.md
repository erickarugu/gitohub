## Gitohub
Gitohub is a simple API that allows you to get GitHub users.

## Installation
```bash
pnpm install
```

## Running the app
```bash
pnpm run start
```

## Running the tests
```bash
pnpm run test
```

## Coverage
```bash
pnpm run coverage
```

- [swagger-ui-express](https://github.com/scottie1984/swagger-ui-express) - swagger ui
